# 🚗 AI-Based ADAS System

## Overview
This project implements an AI-based Advanced Driver Assistance System (ADAS) using deep learning and computer vision.

## Features
- 🚘 Object Detection using YOLOv8
- 🛣️ Lane Detection
- 🚨 Collision Warning System
- 📊 Real-time FPS monitoring
- 🎨 Modern Dashboard UI (Streamlit)

## Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- Streamlit

## How to Run
```bash
pip install streamlit opencv-python ultralytics numpy
streamlit run adas_dashboard.py
```

## Future Scope
- Hardware integration (Raspberry Pi / Jetson Nano)
- Real-time camera input
- Deep learning lane detection
